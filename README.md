# 🇪🇸 UAV Safety Guide Spain | Pro Edition

[![Deploy to GitHub Pages](https://github.com/faustogm-beep/spain-uav-safety/actions/workflows/deploy.yml/badge.svg)](https://github.com/faustogm-beep/spain-uav-safety/actions/workflows/deploy.yml)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Una plataforma de alto rendimiento para pilotos de UAS en España, diseñada para ofrecer una alternativa más rápida, intuitiva y estética que las herramientas oficiales.

## 🚀 Características Premium

- **🗺️ Mapas de Alta Velociad**: Integración directa con los servicios ArcGIS REST de **ENAIRE**.
- **🌤️ Meteorología Avanzada**: Datos en tiempo real vía **AEMET OpenData API**.
- **💎 Diseño Glassmorphism**: Interfaz moderna, minimalista y optimizada para uso en campo.
- **📱 PWA Ready**: Instalable en dispositivos móviles para acceso offline y mayor rendimiento.
- **⚖️ Soporte Legal**: Acceso directo a formularios de comunicación previa para el Ministerio del Interior.

## 🛠️ Tecnologías

- **Core**: [Vite](https://vitejs.dev/) + [TypeScript](https://www.typescriptlang.org/)
- **Mapping**: [Leaflet](https://leafletjs.com/) + [Esri Leaflet](http://esri.github.io/esri-leaflet/)
- **Styling**: Vanilla CSS con variables avanzadas y efectos de desenfoque.
- **CI/CD**: GitHub Actions para despliegue automático en GitHub Pages.

## 📦 Instalación Local

```bash
# Clonar el repositorio
git clone https://github.com/faustogm-beep/spain-uav-safety.git

# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev
```
